1 ¿Qué es un quiz interactivo con Pygame?

Un quiz interactivo es un programa que lanza preguntas al usuario, registra su respuesta y da retroalimentación inmediata. Con Pygame puedes añadir imágenes de fondo, efectos de sonido y animaciones.

Flujo del programa: Aparece la pregunta con 4 opciones → el usuario pulsa A/B/C/D → suena acierto o error y se muestra si era correcta → siguiente pregunta → pantalla final con puntuación.

🧩 Diseña el contenido del quiz

Antes de programar, el equipo diseña las preguntas. Una buena pregunta tiene una respuesta correcta clara y tres distractores creíbles pero incorrectos.

Reglas de oro
  • Una sola idea por pregunta.
  • Autosuficiente. Se entiende sin contexto previo.
  • Evita el "no" en la pregunta.
  • Distractores plausibles pero incorrectos para quien sabe el tema.
  • La respuesta correcta no destaca por ser más larga.
  • Varía las posiciones de la respuesta correcta.
✅ Pregunta bien formulada
ODS 4: Educación de calidad
Según la UNESCO, ¿cuántos niños y niñas en el mundo no tienen acceso a la educación primaria?
A. Unos 20 millones
B. Más de 250 millones
C. Unos 5 millones
D. Unos 80 millones
Los distractores son plausibles. La respuesta correcta no destaca por el formato.
❌ Pregunta mal formulada
Evitar este estilo
¿Cuál de estas afirmaciones NO es correcta sobre el ODS 4?
A. Busca mejorar la educación
B. Fue aprobado en 2015
C. Solo afecta a países en desarrollo
D. Incluye metas para 2030
El "NO" confunde y no evalúa bien el conocimiento real.

💻 Programa tu quiz

Aquí tienes la plantilla base completa y explicada. Entender el código es más importante que copiarlo.

Antes de empezar: instala Pygame si no lo tienes. En la terminal: pip install pygame
¿Cómo está organizado el programa?
1. Importaciones y configuración → Pygame, pantalla, colores y fuentes.

2. Archivos opcionales con os.path.exists() → Imagen, logo y sonidos se cargan solo si el archivo existe. Si no están, el programa arranca igual.

3. Lista de preguntas → Diccionarios con texto, opciones y respuesta correcta.

4. Función dibujar_texto() → Texto largo con ajuste automático de línea.

5. Bucle principal while True → Mantiene la ventana abierta y detecta pulsaciones.

6. Pantalla final → Muestra la puntuación.

import pygame, sys, os
pygame.init(); pygame.mixer.init()
ANCHO, ALTO = 800, 600
pantalla = pygame.display.set_mode((ANCHO, ALTO))
pygame.display.set_caption("Quiz ODS — Vuestro título aquí")
BLANCO=(255,255,255); VERDE=(46,160,67); ROJO=(200,50,50)
AZUL=(50,100,200); FONDO_COLOR=(20,40,80)
fuente_grande=pygame.font.SysFont("Arial",32,bold=True)
fuente_normal=pygame.font.SysFont("Arial",22)
fuente_pequeña=pygame.font.SysFont("Arial",18)
if os.path.exists("fondo.jpg"):
    fondo=pygame.image.load("fondo.jpg"); fondo=pygame.transform.scale(fondo,(ANCHO,ALTO))
else: fondo=None
if os.path.exists("logo_ods.png"):
    logo=pygame.image.load("logo_ods.png"); logo=pygame.transform.scale(logo,(100,100))
else: logo=None
if os.path.exists("acierto.wav"): sonido_acierto=pygame.mixer.Sound("acierto.wav")
else: sonido_acierto=None
if os.path.exists("error.wav"): sonido_error=pygame.mixer.Sound("error.wav")
else: sonido_error=None
preguntas = [
    {"texto":"¿Cuántos ODS tiene la Agenda 2030 de la ONU?",
     "opciones":["A) 10","B) 17","C) 21","D) 8"],"correcta":"B"},
    # ── Añadid vuestras preguntas aquí ──
]
def dibujar_texto(sup,texto,fuente,color,x,y,max_ancho=700):
    palabras=texto.split(); linea,lineas="",[]
    for p in palabras:
        prueba=linea+p+" "
        if fuente.size(prueba)[0]>max_ancho: lineas.append(linea); linea=p+" "
        else: linea=prueba
    lineas.append(linea)
    for i,l in enumerate(lineas):
        sup.blit(fuente.render(l,True,color),(x,y+i*fuente.get_linesize()))
indice,puntuacion=0,0; esperar=False
resultado_texto,resultado_color="",BLANCO
reloj=pygame.time.Clock()
while True:
    if fondo: pantalla.blit(fondo,(0,0))
    else: pantalla.fill(FONDO_COLOR)
    if logo: pantalla.blit(logo,(ANCHO-120,20))
    if indice>=len(preguntas):
        dibujar_texto(pantalla,f"¡Completado! {puntuacion}/{len(preguntas)} correctas",fuente_grande,BLANCO,80,220)
        dibujar_texto(pantalla,"Pulsa ESC para salir",fuente_normal,BLANCO,240,300)
    else:
        p=preguntas[indice]
        dibujar_texto(pantalla,f"Pregunta {indice+1} de {len(preguntas)}",fuente_pequeña,AZUL,50,30)
        dibujar_texto(pantalla,p["texto"],fuente_grande,BLANCO,50,80)
        for i,op in enumerate(p["opciones"]):
            dibujar_texto(pantalla,op,fuente_normal,BLANCO,80,220+i*60)
        dibujar_texto(pantalla,"Pulsa A / B / C / D para responder",fuente_pequeña,(180,180,180),50,490)
        if resultado_texto:
            dibujar_texto(pantalla,resultado_texto,fuente_grande,resultado_color,80,530)
            dibujar_texto(pantalla,"  (pulsa cualquier tecla para continuar)",fuente_pequeña,(180,180,180),80,565)
    for evento in pygame.event.get():
        if evento.type==pygame.QUIT: pygame.quit(); sys.exit()
        if evento.type==pygame.KEYDOWN:
            if evento.key==pygame.K_ESCAPE: pygame.quit(); sys.exit()
            if not esperar and indice<len(preguntas):
                tecla={pygame.K_a:"A",pygame.K_b:"B",pygame.K_c:"C",pygame.K_d:"D"}.get(evento.key)
                if tecla:
                    if tecla==preguntas[indice]["correcta"]:
                        if sonido_acierto: sonido_acierto.play()
                        resultado_texto="✓  ¡Correcto!";resultado_color=VERDE;puntuacion+=1
                    else:
                        if sonido_error: sonido_error.play()
                        cx=preguntas[indice]["opciones"]["ABCD".index(preguntas[indice]["correcta"])]
                        resultado_texto=f"✗  Incorrecto — era: {cx}";resultado_color=ROJO
                    esperar=True
            elif esperar: indice+=1;resultado_texto="";esperar=False
    pygame.display.flip(); reloj.tick(30)

🎬 Vídeo en esquina con OpenCV

Vais a añadir el vídeo como elemento decorativo en una esquina de la pantalla, manteniendo la imagen de fondo que ya habéis creado. El trabajo de diseño que ya tenéis no se toca.

¿Cómo queda la pantalla?

El resultado es una composición con tres capas dibujadas en orden:

Capa 1 — fondo.jpg: vuestra imagen de fondo ocupa toda la pantalla, igual que antes.

Capa 2 — vídeo en esquina: el vídeo se reproduce en bucle en la esquina inferior derecha, en un recuadro de 200×150 px.

Capa 3 — logo, texto y preguntas: todo lo demás se dibuja encima, sin cambios.
El vídeo se puede colocar en cualquier esquina cambiando sus coordenadas. Por defecto va a la esquina inferior derecha para no interferir con las preguntas.

def obtener_fotograma_esquina(cap):
    """Lee el siguiente fotograma y lo escala al tamaño del recuadro.
    Si el vídeo termina, vuelve al inicio automáticamente."""
    ret, frame = cap.read()
    if not ret:
        cap.set(cv2.CAP_PROP_POS_FRAMES, 0)
        ret, frame = cap.read()
    frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)

    return pygame.surfarray.make_surface(frame.transpose(1, 0, 2))

        dibujar_texto(pantalla, "Pulsa ESC para salir",
            fuente_normal, BLANCO, 240, 300)
    else:
        p = preguntas[indice]
        dibujar_texto(pantalla,
            f"Pregunta {indice+1} de {len(preguntas)}",
            fuente_pequeña, AZUL, 50, 30)
        dibujar_texto(pantalla, p["texto"], fuente_grande, BLANCO, 50, 80)
        for i, op in enumerate(p["opciones"]):
            dibujar_texto(pantalla, op, fuente_normal, BLANCO,
                          80, 220 + i * 60)
        dibujar_texto(pantalla,
            "Pulsa A / B / C / D para responder",
            fuente_pequeña, (180, 180, 180), 50, 490)
        if resultado_texto:
            dibujar_texto(pantalla, resultado_texto,
                fuente_grande, resultado_color, 80, 530)
            dibujar_texto(pantalla,
                "  (pulsa cualquier tecla para continuar)",
                fuente_pequeña, (180, 180, 180), 80, 565)

    for evento in pygame.event.get():
        if evento.type == pygame.QUIT: pygame.quit(); sys.exit()
        if evento.type == pygame.KEYDOWN:
            if evento.key == pygame.K_ESCAPE: pygame.quit(); sys.exit()
            if not esperar and indice < len(preguntas):
                tecla = {
                    pygame.K_a: "A", pygame.K_b: "B",
                    pygame.K_c: "C", pygame.K_d: "D"
                }.get(evento.key)
                if tecla:
                    if tecla == preguntas[indice]["correcta"]:
                        if sonido_acierto: sonido_acierto.play()
                        resultado_texto  = "✓  ¡Correcto!"
                        resultado_color  = VERDE
                        puntuacion      += 1
                    else:
                        if sonido_error: sonido_error.play()
                        cx = preguntas[indice]["opciones"][
                            "ABCD".index(preguntas[indice]["correcta"])]
                        resultado_texto = f"✗  Incorrecto — era: {cx}"
                        resultado_color = ROJO
                    esperar = True
            elif esperar:
                indice += 1; resultado_texto = ""; esperar = False

    pygame.display.flip()
    reloj.tick(30)
¿Qué hace cada línea nueva?
1
import cv2
Importa OpenCV. Sin esta línea el programa no sabe qué es cv2 y daría error.
2
VIDEO_ANCHO, VIDEO_ALTO = 200, 150
Define el tamaño del recuadro donde se mostrará el vídeo. Podéis cambiar estos valores para hacerlo más grande o más pequeño.
3
VIDEO_X = ANCHO - VIDEO_ANCHO - 10
Calcula la posición horizontal del recuadro. Restando el ancho del vídeo y 10 px de margen al ancho total, el vídeo queda pegado a la derecha.
4
cv2.VideoCapture("fondo_ods.mp4")
Abre el archivo de vídeo para leerlo fotograma a fotograma.
5
cap.set(cv2.CAP_PROP_POS_FRAMES, 0)
Cuando el vídeo termina, lo rebobina al fotograma 0 para que vuelva a empezar en bucle.
6
cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
OpenCV guarda los colores en orden BGR y Pygame los necesita en RGB. Esta línea hace el cambio.
7
pygame.draw.rect(..., 2)
Dibuja un marco de 2 px alrededor del recuadro del vídeo. Es opcional: si no os gusta, simplemente borrad esas dos líneas.
Podéis cambiar la posición del vídeo ajustando VIDEO_X y VIDEO_Y. Por ejemplo, para ponerlo en la esquina superior izquierda: VIDEO_X = 10 y VIDEO_Y = 10.
Checklist — antes de pasar a mejoras

  • El archivo se llama exactamente fondo_ods.mp4 y está en la carpeta del proyecto
  • Hemos añadido import cv2 al inicio
  • Hemos añadido el bloque de vídeo después del bloque de fondo.jpg (no en su lugar)
